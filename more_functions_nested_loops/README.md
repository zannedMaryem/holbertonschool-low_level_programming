
# More Functions and Nested Loops

## Description

This directory contains C programs demonstrating functions and nested loops concepts.

## Files

| File | Description |
|------|-------------|
| `0-isupper.c` | Check if character is uppercase |
| `1-isdigit.c` | Check if character is digit |
| `2-mul.c` | Multiply two integers |
| `3-print_numbers.c` | Print numbers 0-9 |
| `4-print_most_numbers.c` | Print numbers excluding 2 and 4 |

## How to Compile

```bash
gcc -Wall -Werror -Wextra -pedantic *.c -o output
```
